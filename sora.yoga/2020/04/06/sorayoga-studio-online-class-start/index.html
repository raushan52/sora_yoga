SORAYOGA studio ONLINE CLASS START

Posted 2020年4月6日2020年4月7日 SORAYOGA Studio

いつもSORAYOGA studioをご愛顧いただき誠にありがとうございます。

新型コロナウィルス感染症の広がりにより不安な日々が続いてますが、皆様お変わりございませんでしょうか？

先日4月1日には、こんな状況の中、六甲道に2号店がオープンしました。自分で言うのもおかしいですが、垂水店とはまた違う素敵なスタジオが出来ました！

日本だけでなく、世界にダメージを与え続けているこの最悪な状況も全て宇宙の法則では決まっていることなのですが、

神様は私達に何を教えようとしているのか、

どんな試練を与えたいのか、自問自答をしながら決めたオープンでした。

私は先が読めなさすぎて逆に燃えてきましたがね！笑

今こそ私達人間が本当に大切なことは何か、本当の生きる意味を知り、幸福とは何なのか、を培ってきたヨーガの知恵を生かしていく時なのではないでしょうか。

この暗いトンネルを抜けた時、どのタイミングでも皆様がヨガを再開、または新しく始められるように六甲道スタッフ一同笑顔で皆様をお迎えしますので、その時はよろしくお願いしますね！

なので、六甲道スタジオは来週より今月いっぱい一旦クローズします！オープンしてすぐクローズって、もうこれ以上の最悪の状況は今後ないでしょうね！笑

さてさて、話は変わりますが、

垂水スタジオでは今にも発令される兵庫県の緊急事態宣言により、本日スタッフとミーティングを開き、来週から全てのクラスをオンラインクラスへと変更することを決定致しました。

昨日よりスタートしたオンラインお試しクラスですが、今週はあと、8日(水)10:00〜のヒロミ先生のピラティスBASICと、9日(木)14:30〜のヒトミ先生のゆるめるヨガがお試しで無料で受けれます。

受講方法はまず、「ZOOM」というアプリをインストールしてください。ネットが繋がればスマホでもPCでも参加できます。

お試しを申し込みしたい方は直接メールか私にSNSからでもいいのでメッセージをください。IDとパスワードを発行しますので、そちらを入力していただくと簡単に参加することができます。

ZOOM登録方法

https://zoom-kaigi.com/zoom-touroku/

講師は15分前より待機しております。

〈お試し後の4月オンラインクラスへの参加方法〉

今までと同じようにHP内スケジュールからご予約をお願いします。

ご新規様も受け付けております。

どのクラスも(キッズイングリッシュヨガ以外)GW明けまで¥5500(税込)で受講できます。

お支払いは事前振込で今週中(12日まで)にお伝えする口座に振り込みをお願いします。

振込手数料はご負担ください。

ご入金の確認後お試し時に使用していた番号とは違うIDとパスワードをお伝えします。

返信にお時間を頂戴しますが、質問や問い合わせはいつでも受け付けております。

ご不便をおかけしますが、ご理解いただきますようよろしくお願いします。

皆様にはいつもご尽力いただき感謝しております。

オンラインクラスで皆様と繋がりまた会える日まで一緒に乗り越えていきましょう！
